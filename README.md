# IETC-KotConnect
Ce repo contient le projet "KotConnect" pour mon cours " Projet de développement web " pour l'année académique 2025-2026 à l'IETC. KotConnect est une application qui met en lien les étudiants et les propriétaires de KOTs / studios qui sont disponibles à la location. ( application de présentation de biens et gestion de locations)
